# gitHub查看项目树的插件Octotree

# 作用

当我们在查看gitHub中的项目时对项目不能有一个清晰的了解，跳转也颇为麻烦，因此可以借助此工具，让项目更加清晰明了，查看更方便快捷

# 插件示例图

插件crx文件地址

# 使用方法

下载插件后打开chrome浏览器 -> 右上角菜单 -> 更多工具 -> 扩展程序 -> 右上角的扩展程序打开 -> 将文件拖入即可
